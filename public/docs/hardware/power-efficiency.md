# 電力効率とバッテリーのトレードオフ

コア数が多いことは、必ずしも全てのユーザーにメリットをもたらさない。コア構成の違いについては、[コア構成の物理的特性](./core-architecture.md)を参照。

## バッテリー消費の比較

| 項目 | **M4 Pro** | **M5** |
| --- | --- | --- |
| **P-core数** | 6〜8コア | 4コア |
| **E-core数** | 4コア | 6コア |
| **アイドル時の消費電力** | やや高い | 低い |
| **バッテリー持続時間** | 標準的 | **長い** |

## P-coreの電力消費特性

P-coreは電力を多く消費する。コア数が多いチップは、アイドル時（何もしていない時）のベース消費電力もわずかに高い。

### 電力消費の内訳

- **P-core 1コアあたり**: 高負荷時で数W〜10W以上
- **E-core 1コアあたり**: 高負荷時でも1W以下
- **アイドル時**: P-coreが多いほど、ベース消費電力が増える

## 発熱と静音性

物理的なコア数が多いと発熱量が増え、冷却ファンが回りやすくなる。

### M4 Proの特性

- **高負荷時**: ファンが回りやすく、発熱も大きい
- **静音性**: 低負荷時は静かだが、高負荷時はファン音が目立つ
- **バッテリー**: 標準的な持続時間（10〜12時間程度）

### M5の特性

- **高効率コア（E-core）を多めに配置**: 低負荷時の静音性と圧倒的な電池持ちを実現
- **発熱**: P-coreが少ないため、発熱量が抑えられる
- **バッテリー**: 長い持続時間（12〜15時間以上）

## モビリティ重視の選択

外出先での作業、カフェでの作業、長時間の会議など、**バッテリーの持続時間と静音性を重視する**場合、M5が有利となる。
